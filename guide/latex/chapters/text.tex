\chapter[Guidelines for writing text]{Introductory guidelines for writing text}

When writing the text of your notes, there are a few guidelines to keep in mind:

\section{Spacing}

\LaTeX{} generally ignores additional spaces and multiple line breaks in the source code.
To create a new paragraph, simply leave an empty line in the source code.
If you really need to force a line break without starting a new paragraph, you can use the \verb|\\| command at the end of the line.

Example (demonstrative, bad practice):
\begin{lstlisting}[language=TeX]
This is     a sentence with extra  spaces.\\
Here is a new line without starting a new paragraph. Very
long sentence that continues on the next line in the source
code but will be formatted correctly in the output. Another
sentence.

New paragraph.
\end{lstlisting}
\begin{quote}
This is     a sentence with extra  spaces.\\
Here is a new line without starting a new paragraph. Very long sentence that continues on the next line in the source code but will be formatted correctly in the output. Another sentence.

New paragraph.
\end{quote}

Example (good practice):
\begin{lstlisting}[language=TeX]
This is a sentence with no extra spaces.

Here is a new line without starting a new paragraph.
Very long sentence that continues on the next line in the
source code but will be formatted correctly in the output.
Another sentence.

New paragraph.
\end{lstlisting}
\begin{quote}
This is a sentence with no extra spaces.

Here is a new line without starting a new paragraph.
Very long sentence that continues on the next line in the
source code but will be formatted correctly in the output.
Another sentence.

New paragraph.
\end{quote}

In short,
\begin{itemize}
    \item Prefer leaving blank lines to create new paragraphs over using \verb|\\|.
    \item Make frequent line breaks in the source code to improve readability and the localization of changes and errors. \LaTeX{} will treat the line breaks in the source code as spaces, so they won't affect the formatting of the output.
    \item Use extra spaces or extra blank lines for readability when needed.
\end{itemize}

\section{Special characters}

Keep in mind the following:
\begin{itemize}
    \item \textbf{Double quotes} should be written as \verb|``| (backticks) for opening and \verb|''| (apostrophes) for closing, to ensure proper typographical formatting.
    Example: \verb|``quote''| will produce ``quote'' (correct), while \verb|''quote''| will produce ''quote'' (wrong) and \verb|"quote"| will produce "quote" (wrong).
    \item The same is true for \textbf{single quotes}: \verb|`| (backtick) for opening and \verb|'| (apostrophe) for closing.
    Example: \verb|`quote'| will produce `quote' (correct).
    \item Add \verb|\| after the \textbf{period of an abbreviation} which is not at the end of a sentence, to prevent \LaTeX{} from adding extra space after it.
    Example: \verb|e.g.\ example| will produce e.g.\ example (correct), while \verb|e.g. example| will produce e.g. example (wrong, notice the extra space).
    \item The \textbf{non-breaking space} character \verb|~| can be used to prevent a line break between two words or elements.
    Example: \verb|Dr.~Smith| will produce Dr.~Smith, ensuring that ``Dr.'' and ``Smith'' stay on the same line.
\end{itemize}

\section{Lists}

Bullet lists can be created using the \texttt{itemize} environment, while numbered lists can be created using the \texttt{enumerate} environment.

Example:
\begin{lstlisting}[language=TeX]
\begin{itemize}
    \item First item
    \begin{itemize}
        \item Nested item
        \begin{enumerate}
            \item Nested numbered item
            \item Another nested numbered item
        \end{enumerate}
        \item Nested item
    \end{itemize}
    \item Last item
\end{itemize}
\end{lstlisting}
\begin{quote}
\begin{itemize}
    \item First item
    \begin{itemize}
        \item Nested item
        \begin{enumerate}
            \item Nested numbered item
            \item Another nested numbered item
        \end{enumerate}
        \item Nested item
    \end{itemize}
    \item Last item
\end{itemize}
\end{quote}
