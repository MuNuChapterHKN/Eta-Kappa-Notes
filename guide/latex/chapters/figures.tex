If you open the \texttt{.tex} file, you will notice that I have inserted the \texttt{\textbackslash hfill} command between each graph. This is to prevent \LaTeX{} from optimizing the space by fragmenting the bulleted list below between the graphs. Don't believe me? Try it and see what chaos it creates.

Use the labels when you need to refer to formulas, figures, tables, and pages in your text.
Here are the main ways you can reference an object with an associated label:
\begin{enumerate}
    \item \texttt{\textbackslash ref\{fig:esempio\_geogebra\}}: To refer to the number associated with the label, such as a figure, table, section, etc. (example: Fig.\!~\ref{fig:esempio_geogebra}).
    \item \texttt{\textbackslash pageref\{fig:esempio\_geogebra\}}: To refer to the page number where the labeled object appears (example: page~\pageref{fig:esempio_geogebra}).
    \item \texttt{\textbackslash eqref\{fig:esempio\_geogebra\}}: To refer to an equation, if the label has been applied to an equation environment (example: eq.\!~\eqref{fig:esempio_geogebra}).
\end{enumerate}
